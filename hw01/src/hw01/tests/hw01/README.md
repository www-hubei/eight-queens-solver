## 项目概述
本项目通过 Claude Code 作为AI协作工具，完成八皇后问题的**回溯法实现**、**单元测试覆盖**、**代码健壮性优化**全流程，严格遵循Python工程化规范，核心目标是验证AI工具在编程项目中的协作效率。

## 技术方案
### 核心算法：回溯法
1. **逐行放置**：从第0行开始，逐行尝试在每一列放置皇后。
2. **冲突校验**：通过3个集合分别记录已占用列、主对角线（行-列）、副对角线（行+列），实现O(1)时间复杂度的冲突判断。
3. **递归回溯**：放置合法则递归下一行，不合法则回溯撤销当前选择，直至遍历所有可能。
